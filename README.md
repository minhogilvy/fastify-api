# Fastify Node API Structure

Showcasing Fastify in a modern Node.js project.

### Run the app

- `docker compose up -d`

## Technologies

### Environment

- `Node.js`
- `Typescript`
- `SWC`
- `pnpm`

### Libraries

- `Fastify`
- `Pino`
- `Typebox`
- `Kysely`
- `Paseto`
- `BullMQ`
- `Vitest`
- `ESLint`

### Services

- `Postgres`
- `Mongodb`
- `Redis`

### Putting it all together

- `Docker`
